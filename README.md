## About

HabibiChill is a free, private, offline-first Progressive Web App designed to help Muslims manage anger using authentic Islamic teachings. It combines Quranic guidance, Prophetic ﷺ traditions, and modern emotional regulation techniques into an accessible, mobile-first experience.

> No account required. No data leaves your device. 100% client-side.

## Features

### 🔥 Emergency Anger Protocol
- "CHILL" button with 5-step Sunnah-based calming flow
- **Inline tools**: silence timer, breathing exercise, wudu guide — all embedded within the flow
- Step skip/complete tracking with verification
- Seek Refuge (A'ūdhu billāh) with audio recitation (Alafasy)
- Auto-journal with tactics used and duration tracking

### 📖 Quran & Sunnah Hub
- Dual-action interface: Listen (audio player) + Read (verse-by-verse reader)
- 11 themed surah collections for calmness, protection, and patience
- Anger-relevant ayah collections with Arabic, transliteration, and translation
- 8 authenticated hadiths on anger with full scholarly references
- Bookmarking system for ayahs, hadiths, and duas
- Personalized recommendations based on mood and anger history

### 🧠 Mood Tracking
- Multi-daily mood check-ins with context notes
- Today's mood timeline chart
- 14-day trend analysis
- Time-of-day mood patterns (morning/afternoon/evening/night)
- Mood-anger correlation insights
- Editable mood entries

### 📿 Dhikr Counter
- SubhanAllah (33), Alhamdulillah (33), Allahu Akbar (34) with progress ring
- Auto-advance between phases with completion hadith

### 💧 Wudu Guide
- 11-step interactive tutorial with images and sunnah references

### 🤫 Silence Timer
- Countdown with rotating motivational hadiths and Quran audio

### 🌊 Breathing Exercise
- 4-4-6-2 technique with dhikr integration

### 📓 Anger Journal
- Trigger categories, intensity scale, tactics tracking, reflections
- Expandable recent entries on home with full details

### 🧭 Situation Guides
- 6 real-life scenarios: Marriage, Online, Family, Workplace, Parenting, Traffic

### 📚 Learn Tab
- 8 structured lessons from basics to advanced (Imam Ghazali)
- Completion tracking with localStorage persistence

### 📊 Analytics & Progress
- Weekly anger patterns, intensity trends, trigger breakdown
- Achievement badges and Sabr Points system
- Data export (JSON/CSV) and import

### 📝 Blog & Guides
- SEO-optimized blog with category/tag filtering and URL search params
- Comprehensive guides page with accordion sections
